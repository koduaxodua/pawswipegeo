

# 🐾 PawSwipe — ტინდერი მიუსაფარი ძაღლებისთვის

## კონცეფცია
ტინდერის სტილის მობაილ-ფერსტ ვებ აპლიკაცია, სადაც მომხმარებლები სვაიპით ეცნობიან მიუსაფარ ძაღლებს და ინახავენ მოწონებულებს. Liquid glass დიზაინი თბილი ფერებით.

## მთავარი გვერდები

### 1. მთავარი — სვაიპ კარტები
- ტინდერის სტილის კარტები ძაღლის ფოტოთი, სახელით, ასაკით, ლოკაციით
- სვაიპი მარჯვნივ = ❤️ (მოწონება), მარცხნივ = ❌ (გამოტოვება)
- ღილაკებიც იქნება სვაიპის გარდა
- კარტზე დაჭერით — დეტალური ინფო (ჯიში, ხასიათი, ჯანმრთელობა, მიმკედლებლის კონტაქტი)

### 2. მოწონებულები 💕
- დალაიქებული ძაღლების სია
- თითოეულზე: ფოტო, სახელი, მიმკედლებლის ტელეფონი, ლოკაცია, დეტალები
- localStorage-ში შენახვა (რეგისტრაციის გარეშე)

### 3. ძაღლის დამატება ➕
- ფორმა: ფოტოს ატვირთვა (URL), სახელი, ასაკი, ჯიში, ხასიათი, ჯანმრთელობა, ლოკაცია, საკონტაქტო ნომერი
- მარტივი ვალიდაცია

### 4. Terms / პირობები
- ქართული კანონმდებლობის შესაბამისი პირობები (ცხოველთა დაცვის კანონი, პერსონალური მონაცემები)
- პასუხისმგებლობის შეზღუდვა

## დიზაინი
- **Liquid glass** სტილი — გამჭვირვალე, ბლურიანი კარტები, რბილი ჩრდილები
- **თბილი პალიტრა**: ნარინჯისფერი/ატამისფერი აქცენტები, კრემისფერი ფონი
- 🐾 ძაღლური ლოგო და აიქონები
- მობაილ-ფერსტ, ადაპტირებული დესკტოპზეც
- ქართული ტექსტი

## ტესტ მონაცემები
- 6-8 სატესტო ძაღლი ინტერნეტის ფოტოებით
- ქართული სახელები, თბილისის ლოკაციები, ფეიკ ტელეფონის ნომრები

## ტექნიკური
- მთლიანად ფრონტენდზე (localStorage) — ბექენდის გარეშე, Digital Ocean-ზე მარტივად აიტვირთება
- React + Tailwind + Framer Motion (სვაიპ ანიმაციები)
- React Router — ნავიგაცია გვერდებს შორის
- ქვედა ნავიგაციის ბარი (მობაილ სტილი)
